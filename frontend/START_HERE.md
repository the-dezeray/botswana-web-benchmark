# 🚀 Start Here

## Run the Dashboard

```bash
pnpm dev
```

Then open: **http://localhost:5173**

---

## What You'll See

✨ **Minimal, clean scientific report design**

### Sections
1. 📊 Metric cards (4 key stats)
2. 📈 Performance scores (bar chart)
3. ⏱️ LCP times (bar chart)
4. 💾 Page sizes (bar chart)
5. 🏢 Industry averages (bar chart)
6. 📋 Full breakdown table
7. 🍩 Resource breakdowns (donut charts)

### Design Style
- Clean, flat design
- Subtle borders and spacing
- Performance-based colors
- Professional typography
- Responsive layout

---

## Update Data

After running new audits:

```bash
pnpm update-data
```

Or manually:
```bash
copy ..\results_quick.json public\results_quick.json
```

## Build for Production

```bash
pnpm build
pnpm preview
```
